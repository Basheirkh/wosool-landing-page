# Wosool Content Architecture

## What Stays in the Landing Page

The landing page should answer only:

1. What is Wosool?
2. Why does it matter?
3. What makes it different?
4. How fast does it start?
5. Can I trust it?
6. What should I do next?

Rule:
- One sharp heading per section
- One short support line
- Optional 2-3 bullets only
- Animation explains the rest
