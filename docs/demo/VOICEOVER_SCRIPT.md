**Target VO duration:** ~92s of narration + ~26s breathing room for music/SFX

---

## Dialect guardrails for generation

Use these forms consistently; they're the spine of the "Saudi natural" read:

| Formal (avoid) | Saudi (use) | Notes |
|---|---|---|
| لديك | عندك | possession |
| تريد | تبي | want |
| ماذا | وش | what |
| الآن | الحين | now (Hijazi-core) |
| يقوم بـ | يسوّي | does |
| هذا الأمر | كذا | like this / this way |
| جيد جداً | فله · تمام | good / excellent |
| دقيقة واحدة | دقيقة وحدة | one minute (natural) |
| جرّب | جرّب | same, keep it |
| ثمّ | بعدين | then |
| سوف | راح | will |
| فقط | بس | just/only |

**Pacing:** ~2.8–3.2 words per second for natural Arabic read. Don't rush the hook (slower), pick up through conversation arcs (moderate), soften again at the closer.

**Vocal emotion map:**
- Hook: quiet → confident reveal → warm intro
- Setup: matter-of-fact, instructional
- Conversation arcs: energized, demo-showing-off
- Customer: compassionate (the human moment)
- Policy: matter-of-fact
- Sales: warm, "live" energy
- Closer: earned confidence → CTA warmth

**Silence is a tool.** Don't fill every scene. Leave gaps so on-screen motion can breathe, and so the viewer hears UI beats (beeps, cart ding, notification swoosh if we add SFX in phase 7).

### Clip 1 · HOOK · narrator
**File:** `01-hook.mp3` · **Target duration:** ~11s · **Trigger:** scene start of `cold-open` (0:00)

```
[quiet, almost whispered · 0:02]
كل ليلة...

[1s beat]

[slightly firmer]
رسائل من عملاءك... تضيع.

[1.5s beat · cut to bright the-promise scene]

[shift to confident, warm]
من اليوم.
تكلّم متجرك — يسويه.

[1s beat · cut to meet-the-four]

[warmer, presenting]
```

**Pronunciation notes:**
- "كل ليلة" → *kull layla* (not *kulli laylatin* — drop MSA case endings)
- "تضيع" → *tdhee3* (classical pronunciation is fine — "tadhee3" sounds natural)
- "تكلّم متجرك — يسويه" → *takallam matjarak — yisawweeh* (the tagline, enunciate clearly — this is the brand line)
- "أربعة موظفين" → *arba3a muwadhdhafeen* (Saudi uses MSA numerals cleanly)
- "ما ينامون، ما يتعبون" → *ma yinaamoon, ma yit3aboon* (keep the rhythm — this line is half of the brand)

## Delivery checklist for ElevenLabs generation

For each clip:

1. **Prompt the model** with the dialect guardrails (paste the table at the top)
2. **Set voice** to Jeddawi Echo for narrator clips (1–8)
3. **Use a different voice** for M1, M2 (younger male), V1 (female)
4. **Request multiple takes** per clip — emotional tone is the variable ElevenLabs gets wrong most often
5. **Target audio levels:** narrator ~-18 LUFS, character clips ~-16 LUFS (slightly hotter so they cut through music)
6. **Format:** 48kHz mono MP3, 192 kbps
7. **Normalize:** all clips peak-normalized to -1 dBFS, then loudness-matched against narrator track 1

**Pre-render checks per clip:**
- [ ] Correct dialect (no إياكم / سيتم / لديكم sneaking in)
- [ ] Numbers read naturally, not digit-by-digit
- [ ] Brand words (وصول, سلة, واتساب) are enunciated clearly
- [ ] Pacing matches target duration (±1s)
- [ ] Emotional tone matches scene mood
